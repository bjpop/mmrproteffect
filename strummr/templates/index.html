StruMMR

Structure and function effect predictions for missense variants in the human mismatch repair proteins

Features
StruMMR considers MSH2 and MSH6 variants from the InSiGHT database, and aggregates a diverse range of information, including:
  • Structure and function effect prediction for missense variants using the mCSM suite of tools.
  • Interactive visualisation of variants within Pfam domains and within the MutSα 3D crystal structure.
  • In silico pathogenicity predictions from CADD, REVEL, PolyPhen-2, SIFT, and PROVEAN.
  • Key variant metadata including genomic and protein coordinates and population frequency from gnomAD.
  • New pathogenicity predictions using a machine-learning classifier trained on the features computed by the mCSM suite of tools and the InSiGHT class annotations for missense variants.
  • Interactive plots comparing all variant annotation and metadata variables.
  • Visualisation of interatomic interactions for variant residues within the MutSα 3D crystal structure from the Arpeggio tool.
